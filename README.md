### STORIES PROTO

#### PreRequisites:
```
grpc-web protoc plugin: https://github.com/grpc/grpc-web/releases
protoc: https://github.com/protocolbuffers/protobuf/releases
```

#### setup:
```
make setup
```

#### update go proto:
```
make update-go-proto
```

#### update js proto:
```
make update-js-proto
```

#### update all protos:
```
make update-protos
```
